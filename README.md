
# تمپلیت معماری DDD + CQRS

این ریپازیتوری یک **پروژه تستی و تمپلیت معماری** است که با هدف نمایش **ساختار و لایه‌بندی صحیح** در یک پروژه مبتنی بر DDD، CQRS و Clean Architecture ایجاد شده است.

در این تمپلیت:

* تمرکز صرفاً روی **اسکلت و ساختار معماری** است و منطق بیزنسی واقعی پیاده‌سازی نشده
* انتیتی نمونه با نام **Record** فقط به‌عنوان یک **الگوی جایگزین (Placeholder)** استفاده شده است
* کدها برای استفاده در پروژه‌های واقعی نیاز به **تکمیل و شخصی‌سازی** دارند

این پروژه صرفاً برای **یادگیری، تمرین و نمایش معماری** طراحی شده است.
